## 二、ADC采集模块

### 2.1 文件概述

### 2.2 核心函数

#### 2.2.1 Adc_Init() - ADC初始化

**功能**：配置ADC1，通道0和通道1

```c
void Adc_Init(void)
{
    // 1. 使能GPIOA和ADC1时钟
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA |
                           RCC_APB2Periph_ADC1, ENABLE);
    
    // 2. 配置PA0、PA1为模拟输入
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_0 | GPIO_Pin_1;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AIN;
    GPIO_Init(GPIOA, &GPIO_InitStructure);
    
    // 3. ADC复位
    ADC_DeInit(ADC1);
    
    // 4. ADC配置
    ADC_InitStructure.ADC_Mode = ADC_Mode_Independent;
    ADC_InitStructure.ADC_ScanConvMode = DISABLE;
    ADC_InitStructure.ADC_ContinuousConvMode = DISABLE;
    ADC_InitStructure.ADC_ExternalTrigConv = ADC_ExternalTrigConv_None;
    ADC_InitStructure.ADC_DataAlign = ADC_DataAlign_Right;
    ADC_InitStructure.ADC_NbrOfChannel = 1;
    ADC_Init(ADC1, &ADC_InitStructure);
    
    // 5. 使能ADC
    ADC_Cmd(ADC1, ENABLE);
    
    // 6. ADC校准
    ADC_ResetCalibration(ADC1);
    while(ADC_GetResetCalibrationStatus(ADC1));
    ADC_StartCalibration(ADC1);
    while(ADC_GetCalibrationStatus(ADC1));
}
```

#### 2.2.2 Get_Adc() - 获取ADC值

**功能**：获取指定通道的ADC采样值

```c
u16 Get_Adc(u8 ch)
{
    // 配置ADC通道
    ADC_RegularChannelConfig(ADC1, ch, 1, ADC_SampleTime_239Cycles5);
    
    // 启动软件转换
    ADC_SoftwareStartConvCmd(ADC1, ENABLE);
    
    // 等待转换完成
    while(!ADC_GetFlagStatus(ADC1, ADC_FLAG_EOC));
    
    // 返回转换结果
    return ADC_GetConversionValue(ADC1);
}
```

#### 2.2.3 Get_Adc_Average() - 平均滤波

**功能**：多次采样取平均值，提高精度

```c
u16 Get_Adc_Average(u8 ch, u8 times)
{
    u32 temp_val = 0;
    u8 t;
    for(t = 0; t < times; t++) {
        temp_val += Get_Adc(ch);
        delay_ms(5);
    }
    return temp_val / times;
}
```

#### 2.2.4 Lsens_Get_Val() - 获取环境光强

### 3.4 核心函数

#### 3.4.1 INA226_Init() - 初始化

**功能**：配置INA226，写入校准值

```c
void INA226_Init(void)
{
    // 1. I2C 引脚初始化（PB8/PB9，见 myiic.c 中 IIC_Init_226）
    IIC_Init_226();
    
    // 2. 配置寄存器：16次采样，4.128ms转换时间
    INA226_SendData(INA226_ADDR1, CFG_REG, 0x45FF);
    
    // 3. 写入校准值
```

## 八、定时器PWM模块

### 8.1 文件概述

**文件路径**：`Library_2/TIMER/timer.c/h`  
**功能描述**：TIM3 PWM输出配置

### 8.2 核心函数

#### 8.2.1 TIM3_PWM_Init() - PWM初始化

**功能**：配置TIM3_CH1输出PWM

```c
void TIM3_PWM_Init(u16 arr, u16 psc)
{
    // 1. 使能TIM3和GPIOA时钟
    RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM3, ENABLE);
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA, ENABLE);
    
    // 2. 配置PA6为复用推挽输出
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_6;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AF_PP;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(GPIOA, &GPIO_InitStructure);
    
    // 3. TIM3配置
    TIM_TimeBaseStructure.TIM_Period = arr;
    TIM_TimeBaseStructure.TIM_Prescaler = psc;
    TIM_TimeBaseStructure.TIM_ClockDivision = 0;
    TIM_TimeBaseStructure.TIM_CounterMode = TIM_CounterMode_Up;
    TIM_TimeBaseInit(TIM3, &TIM_TimeBaseStructure);
    
    // 4. PWM配置
    TIM_OCInitStructure.TIM_OCMode = TIM_OCMode_PWM1;
    TIM_OCInitStructure.TIM_OutputState = TIM_OutputState_Enable;
    TIM_OCInitStructure.TIM_Pulse = 0;
    TIM_OCInitStructure.TIM_OCPolarity = TIM_OCPolarity_High;
    TIM_OC1Init(TIM3, &TIM_OCInitStructure);
    
    // 5. 使能TIM3
    TIM_Cmd(TIM3, ENABLE);
}
```

## 九、串口通信模块（USART1）

### 9.1 文件概述

工程中 **实际参与链接的 USART1 初始化函数** 为 `uart1_init()`，定义在 **`Library_2/USART2/usart2.c`**，头文件为 **`usart2.h`**。该实现完成：

- GPIO：PA9（TX）、PA10（RX）
- USART1 参数配置、接收中断（`USART_IT_RXNE`）
- `fputc` 重定向至 USART1，便于 `printf` 输出 AT 指令或调试信息

`Library_2/USART/USART.c` 中存在另一套 `USART1_Init()` 命名实现，是否加入工程以 **Keil 工程组内勾选文件** 为准；与 NB-IoT 模块通信时，请保证工程中仅使用一套 USART1 初始化逻辑，避免符号重复或未调用。

### 9.2 核心函数

#### 9.2.1 uart1_init(u32 bound)
