## 第一部分 软件概述

### 1.1 软件简介

本软件是一套基于 NB-IoT（窄带物联网）的智能路灯控制系统嵌入式软件，运行于 **STM32F103 系列**微控制器（常见为 C8T6；当前 Keil 工程文件中的器件选项可能为 C6，交付与烧录时请与实物芯片一致）。软件集成了 NB-IoT 通信、电力监测、智能调光、定时控制、人体感应、OLED 显示等功能，实现路灯智能化管理与远程监控。

本软件通过采集环境光照强度、路灯功率、电流、电压等参数，按预设策略自动调节路灯亮度，支持定时开关与人体感应控制，经 NB-IoT 将数据上传至**云平台（示例实现对接巴法云 bemfa.com TCP）**，并支持远程控制，提升能效管理并利于节能。

### 1.2 软件功能特点

- **智能调光**：根据环境光照强度自动调节路灯亮度，支持4级亮度调节
- **电力监测**：实时监测电压、电流、功率，精度可达毫瓦级
- **定时控制**：支持设置开关灯时间，独立时钟芯片，断电不丢失
- **人体感应**：采用红外传感器检测人体存在，智能开关灯
- **远程通信**：通过NB-IoT模块实现远程监控和控制
- **本地显示**：OLED显示屏实时显示运行状态
- **状态指示**：RGB LED灯带指示不同工作状态

### 1.3 应用领域

- 智慧城市路灯管理
- 园区智能照明系统
- 道路照明节能改造
- 物联网应用开发教学

## 第三部分 功能模块说明

### 3.1 NB-IoT远程通信模块

**功能描述**：
本模块实现了基于M5310A NB-IoT模块的远程通信功能，通过USART1与NB-IoT模块通信，实现路灯状态数据上传和远程控制指令接收。

**主要功能**：
1. NB-IoT模块初始化
2. 网络注册与连接
3. TCP Socket建立
4. 数据上传到云平台
5. 远程控制指令接收

**技术实现**：
- 通信协议：TCP/IP
- 波特率：9600bps
- 云平台：bemfa.com
- 数据格式：`#环境光百分比,亮度档位,功率_W,路灯状态,定时标志*`（与 `Post_data()` 中 `sprintf` 参数顺序一致）

**核心函数**：
- `M5310A_Init()` - NB-IoT模块初始化
- `network_init()` - 网络连接初始化
- `Post_data()` - 数据上传函数

### 3.2 电力监测模块

**功能描述**：
本模块通过INA226传感器实时监测路灯的电压、电流、功率等电力参数，为能耗统计和故障诊断提供数据支持。

**主要功能**：
1. 电压测量（0-26V）
2. 电流测量（0-3.2A）
3. 功率计算（0-83W）
4. 数据实时更新

**技术实现**：
- 传感器：INA226
- 通信接口：软件 I2C（PB8/PB9）；源码中 `INA226_ADDR1` 为 **0x80**（8 位地址书写，等价 7 位 **0x40**），须与模块 A0/A1 焊接一致
- 分流电阻与校准：以原理图为准；头文件注释示例为 **0.1Ω** 与 `CAL_REG = 0x0800` 配套，更改电阻须重算校准字
- 测量精度：取决于芯片与校准（典型规格见 INA226 数据手册）

**核心函数**：
- `INA226_Init()` - 传感器初始化
- `GetPower()` - 获取电力数据

### 3.3 智能调光模块

**功能描述**：
本模块根据环境光照强度自动调节路灯亮度，实现节能环保的目标。

**主要功能**：
1. 环境光强采集（ADC双通道）
2. 4级亮度调节（关闭/一档/二档/三档）
3. 自动调光算法
4. PWM输出控制

**技术实现**：
- 控制方式：TIM3 PWM输出
- 输出引脚：PA6（TIM3_CH1）
- 采样通道：PA0（环境光）、PA1（路灯反馈）
- 调光算法：分段线性映射

**亮度等级**：软件按环境光分段与定时/感应逻辑切换 **0～3 共 4 档**（关闭 + 三档亮度）。具体 PWM 占空比与 `TIM3_PWM_Init(99, 7199)` 及主循环中 `TIM_SetCompare1` 设定有关，以 `main.c` 实现为准；上表若为教学示例，实际比例请对照源码。

### 3.4 定时控制模块

**功能描述**：
本模块通过DS1302实时时钟芯片实现定时开关灯功能，支持用户设置开关灯时间。

**主要功能**：
1. 实时时钟读取
2. 开关灯时间设置
3. 定时判断逻辑
4. 时间校准功能

**技术实现**：
- 时钟芯片：DS1302
- 通信方式：3线串行接口
- 晶振频率：32.768kHz
- 时间格式：BCD码

**时间设置**：
- 开灯时间：可设置时:分
- 关灯时间：可设置时:分
- 支持跨天定时

**核心函数**：
- `Init_DS1302()` - 时钟芯片初始化
- `GetRealTime()` - 获取实时时间
- `SetRealTime()` - 设置实时时间
- `Key_settime()` - 时间校准函数

### 3.5 人体感应模块

**功能描述**：
本模块通过HC-SR501人体红外传感器检测人体存在，实现有人时自动开灯，无人时自动关灯。

**主要功能**：
1. 人体红外检测
2. 有人/无人状态判断
3. 自动开关灯控制

**技术实现**：
- 传感器：HC-SR501（或兼容模块）
- MCU 输入：**PB7**（`Key.h` 中 `HC_SR501`）
- 输出方式：数字量（注意与 `Key_Init` 中上拉/下拉配置及模块输出电平匹配）
- 检测距离、延时：依模块电位器与透镜安装而定（常见约数米量级）

### 3.6 OLED显示模块

**功能描述**：
本模块通过0.96寸OLED显示屏实时显示系统运行状态，方便用户查看和操作。

**主要功能**：
1. 实时时间显示
2. 环境光强显示
3. 亮度等级显示
4. 功率显示
5. 状态指示

**显示内容布局**：
```
┌─────────────────────────┐
│    HH:MM:SS          │  // 第0行：时间
│  环境光强: XX %      │  // 第2行：光强
│  亮度档位: 三档      │  // 第4行：亮度
│XX.X W 状态:正常    │  // 第6行：功率+状态
└─────────────────────────┘
```

**技术实现**：
- 驱动芯片：SSD1306
- 分辨率：128×64
- 通信接口：软件I2C
- 字库：16×16中文字库

**核心函数**：
- `OLED_Init()` - 显示屏初始化
- `OLED_ShowCH()` - 显示中文
- `OLED_ShowNum()` - 显示数字
- `OLED_Clear()` - 清屏函数

### 3.7 本地按键控制模块

**功能描述**：
本模块通过4个物理按键实现本地参数设置功能，包括时间校准、定时时间设置等。

**按键功能定义**（源码宏名）：
| 宏 / 含义 | 功能 | 引脚 |
|------|------|------|
| `KEY_set` | 进入设置模式 / 切换设置项 | PB12 |
| `KEY1_add` | 参数数值增加 | PB13 |
| `KEY1_del` | 参数数值减少 | PB14 |
| `KEY_time` | 时间校准 / 确认保存 | PB15 |

**设置流程**：
1. 按下 `KEY_set` 进入设置模式
2. 通过 `KEY_set` 切换设置项（开灯时、开灯分、关灯时、关灯分）
3. 通过 `KEY1_add` / `KEY1_del` 调整数值
4. 按下 `KEY_time` 确认保存退出

**核心函数**：
- `Key_Init()` - 按键初始化
- `Set_parameters()` - 参数设置函数
- `Key_settime()` - 时间校准函数

## 第四部分 技术特性

### 4.1 通信协议

#### NB-IoT通信协议

**AT指令序列**：
```
AT              // 测试模块响应
AT+CFUN=1       // 开启全功能
AT+CIMI          // 获取IMSI
AT+CGATT=1      // 附着网络
AT+CGATT?        // 查询附着状态
AT+NSOCR="STREAM",6,0,2  // 创建TCP Socket
AT+NSOCO=0,bemfa.com,8344  // 连接服务器
AT+NSOSD=0,1,"cmd=2&uid=...&topic=...&msg=..."  // 发送数据
```

#### 数据上传格式

**格式**：`#环境光%,亮度档位,功率_W,路灯状态,定时标志*`

**示例**：`#35,2,15.5,0,1*`

**字段说明**：
| 序号 | 说明 | 示例 |
|------|------|------|
| 1 | 环境光强度百分比（0–100） | 35 |
| 2 | 亮度档位 | 2 |
| 3 | 功率（W，INA226 毫瓦/1000） | 15.5 |
| 4 | 路灯状态标志 `LED_state` | 0 |
| 5 | 定时时段标志 `Timed_flag`（0=否，1=是） | 1 |

## 第五部分 操作说明

### 5.1 系统初始化

系统上电后自动执行初始化流程：
1. 系统时钟配置
2. 各外设模块初始化
3. NB-IoT模块连接网络
4. OLED显示初始化信息
5. 进入主循环

### 5.2 时间校准操作

1. 按下 `KEY_time` 进入时间校准模式
2. OLED 显示“校准时间”等提示（以实际字库与界面为准）
3. 显示当前时间
4. 再次按下 `KEY_time` 切换到时、分、秒
5. 使用 `KEY1_add` / `KEY1_del` 调整数值
6. 调整完成后按下 `KEY_time` 保存退出

### 5.3 定时时间设置

1. 按下 `KEY_set` 进入设置模式
2. OLED 显示“设置开灯时间”等提示
3. 使用 `KEY_set` 切换设置项
4. 使用 `KEY1_add` / `KEY1_del` 调整数值
5. 全部设置完成后按下 `KEY_time` 保存退出
